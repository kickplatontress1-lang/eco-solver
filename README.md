# Ecological solver

Телеграм бот на Python для решения экологических проблем
Чтобы запустить этого телеграм бота нужно скачать **telebot** и сгенерировать токен от **BotFather**.

---

## Установка telebot

Откройте новый терминал и введите это:

**Windows:**
`pip install pyTelegramBotAPI`

**MacOS:**
`pip3 install pyTelegramBotAPI`

## Как получить токен от BotFather

1. Откройте BotFather в телеграм [здесь](https://t.me/BotFather)
2. Запусти бота и введи /newbot
3. Называем бота
4. Копируем токен
